# Shikimori Plus v2.4.4

1. Отображение 4К метки на Shikimori на готовых 4К аниме из библиотек Yanima / ilya1708 / krnzhnr (Переключение библиотеки через меню расширения)

2. Выбор нескольких библиотек, при наведении на 4К плашку будет отображаться наличие 4К у всех библиотек даже если эта библиотека не выбрана

3. Кнопка перехода со страницы аниме Yanima на страницу аниме Shikimori

4. Кнопки перехода со страницы аниме Shikimori на страницу аниме Yanima или Anime 365

5. Плеер Kodik на страницах аниме Shikimori

6. Кликнув на русское или английское название аниме на Shikimori оно скопируется

7. В блоке с информацией все значения можно скопировать

8. Добавлен рейтинг Shikimori, стандартный рейтинг MAL

9. В блоке "На других сайтах" добавлены ссылки на Rutracker и Nyaa, при нажатии на них открывается сайт с результатом аниме страницы Shirimori

---------------------------

⚙️ Установка
Firefox
Зайдите на about:addons ➡️ Установить дополнение из файла ➡️ выберите скачанный "Firefox 2.4.4.zip"

Firefox Android
Зайдите в "настройки" ➡️ О Firefox ➡️ 5 раз кликните на логотип Firefox ➡️ Снова перейдите в "Настройки" ➡️ Установить расширение из файла ➡️ выберите скачанный "Firefox 2.4.4.zip"

Chrome
Зайдите на chrome://extensions/ ➡️ Включите "Режим разработчика" ➡️ Загрузить распакованное расширение ➡️ выберите распакованный "Chrome 2.4.4.zip"
